Proyectos

Barbería Alura

Una pagina web responsive y profesional para una barbería ficticia creada como un proyecto de integración para el curso "Desarrollo Front-End" de Alura Latam, con el objetivo de poner en práctica todos los conocimientos teóricos adquiridos y demostrar una comprensión sólida de los conceptos básicos de HTML5 y CSS3.

Number Guesser

Un juego web sencillo donde el usuario tiene que adivinar el número generado por la máquina, usando las pistas que la misma le proporciona luego de cada intento fallido. Hecho con funcionalidades básicas de HTML5 y CSS3, así como JavaScript para hacer el sitio interactivo y realizar las operaciones logicas necesarias para el correcto funcionamiento del juego.
ADVERTENCIA: Esta pagina web aun no es responsive, debido a que actualmente esta en construccion. Por lo cual, su visionado en dispositivos moviles y tabletas puede generar errores visuales.
